# GradeForge

GradeForge is a Student Grade and Course Tracker designed to manage academic records, calculate GPAs, and organize coursework. The project features custom data structure implementations, benchmarked algorithms, robust persistence systems, and a desktop user interface.

## Core Features

- **Domain Model**: Tracks students, courses, assignments, and GPAs using object-oriented principles.
- **Custom Data Structures**: Implements generic lists, stacks, queues, trees, and hash maps from scratch.
- **Algorithms**: Implements and benchmarks searching and sorting algorithms (Bubble, Selection, Insertion, Merge, Quick, Linear, and Binary Search).
- **Persistence**: Supports importing and exporting student data via CSV and JSON.
- **Design Patterns**: Employs strategy, observer, and factory design patterns.
- **Desktop Interface**: Features a JavaFX-based graphical user interface with data visualization charts.

## Project Structure

- `src/`: Java source files and unit tests.
- `docs/`: Development roadmap and documentation.
